# CV Rock-Paper-Scissors

An AI-powered Rock-Paper-Scissors game that recognizes your hand gestures in real-time. Challenge the computer to a classic duel!

## ✨ Features
- **Gesture Recognition**: Uses MediaPipe to identify Rock, Paper, or Scissors gestures.
- **Interactive UI**: Features an overlay with a timer and real-time score tracking.
- **Randomized AI**: The computer selects a move randomly to challenge you.
- **Real-Time Score**: Tracks wins for both the player and the AI.

## 🚀 Installation

1. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the Game**:
   ```bash
   # Note: Run from the inner directory containing the python file and Resources
   cd rock/rock
   python rock.py
   ```

## 🎮 Controls
- **Gestures**:
  - **Fist**: Rock
  - **Open Palm**: Paper
  - **Two Fingers**: Scissors
- **'S'**: Start the 3-second timer for a new round.
- **'Q'**: Quit the game.
