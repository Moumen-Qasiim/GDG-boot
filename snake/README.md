# CV Snake Game

A camera-controlled Snake game where you lead the snake using your index finger. Built with OpenCV and CVZone.

## ✨ Features
- **Finger Control**: Use your index finger to guide the snake across the screen.
- **Dynamic Growth**: The snake grows longer as it eats "Donuts".
- **Collision Detection**: Game over if the snake hits its own body.
- **High Performance**: Optimized hand tracking using MediaPipe.

## 🚀 Installation

1. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the Game**:
   ```bash
   python snake.py
   ```

## 🎮 Controls
- **Index Finger**: Lead the snake.
- **'R'**: Restart game after Game Over.
- **'Q'**: Quit.
