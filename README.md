# GDG-Boot Game Collection

This repository contains various computer vision and logic-based games.

## How to Run

Instead of installing requirements for each game manually, you can use the unified launcher:

1. Open your terminal in the root of this project.
2. Run the launcher:
   ```bash
   python3 run.py
   ```

The launcher will:
- Automatically create a virtual environment (`.venv`).
- Install all necessary dependencies for all games.
- Provide a menu to select and launch any game.

## Available Games

- **Rock Paper Scissors**: A hand-tracking based game.
- **Snake**: Control the snake with your hand.
- **Ping Pong CV**: Real-time ping pong using computer vision.
- **Tic Tac Toe CV**: Classic Tic Tac Toe with hand gestures.
